================================================================================
                        HOW TO USE THE APP
================================================================================

1. START THE SERVER:
   php artisan serve
   ‚Üí Server will run at http://localhost:8000

2. LOGIN:
   - Go to http://localhost:8000/login
   - Use test account:
     Email: john@example.com
     Password: password

3. CREATE A TRANSACTION:
   - Click "New Transaction" in navbar
   - Select Type (Income or Expense)
   - Choose Category
   - Enter Title, Amount, Date, Payment Method
   - Click "Create Transaction"

4. VIEW TRANSACTIONS:
   - Click "Transactions" in navbar
   - Browse all transactions with pagination
   - Use filters to search
   - Click transaction to view details

5. EDIT/DELETE TRANSACTION:
   - Open transaction detail
   - Click "Edit" or "Delete" button

6. VIEW DASHBOARD:
   - Click "Dashboard" in navbar
   - See financial summary
   - Check charts and recent transactions

7. LOGOUT:
   - Click your name in top-right corner
   - Click "Logout"

================================================================================
                        PRODUCTION DEPLOYMENT
================================================================================

Before deploying:

1. Update .env:
   APP_ENV=production
   APP_DEBUG=false

2. Generate new key:
   php artisan key:generate

3. Run migrations:
   php artisan migrate --force
   php artisan db:seed

4. Optimize:
   php artisan config:cache
   php artisan route:cache
   php artisan view:cache

5. Set proper permissions for storage & bootstrap directories

6. Configure web server (Apache/Nginx) to point public/ directory

7. Enable HTTPS
